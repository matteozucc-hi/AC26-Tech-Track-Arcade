# AC26-Tech-Track-Arcade
Arcade code used in Esri UK's Technical Showcase session 'Enhancing maps and apps with simple Arcade expressions' during the 2026 Annual Conference.

In the repository is Arcade code used in:
- The Map Viewer, for:
   - Labelling
   - Symbology
   - Popup configuration
- Dashboards, for:
   - Data expressions
   - List configuration
- Experience Builder:
   - Bringing data in
   - Dynamic text
   - List widget styles
