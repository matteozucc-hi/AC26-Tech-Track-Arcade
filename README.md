# AC26 Tech Track Arcade

### Context
Arcade code used in Esri UK's Technical Showcase session 'Enhancing maps and apps with simple Arcade expressions' during the 2026 Annual Conference.

A recording of this session can be found [here](https://www.esriuk.com/en-gb/about/events/ac/presentations/2026#enhancing-maps-and-apps-with-simple-arcade-expressions).

Moreover, a plethora of even better resources can be found online. We have collected some [here](https://view.page/p/r51BVI). The Esri arcade-expressions [GitHub repo](https://github.com/Esri/arcade-expressions) is of particular note and contains amazing snippets of code you can repurpose.

### Contents
In this repository is Arcade code used in:
- The Map Viewer, for:
   - Labelling
   - Symbology
   - Popup configuration
- Dashboards, for:
   - Data expressions
   - List configuration
- Experience Builder:
   - Data Import
   - Dynamic text
   - List widget styles

### Data
For our demo, we used data from the following sources:
-  The [Transport for London GIS Open Data Hub](https://gis-tfl.opendata.arcgis.com/)
- The [TfL Open Data site](https://tfl.gov.uk/info-for/open-data-users/our-open-data)
- Premium data for points of interest and railways from the Ordnance Survey
